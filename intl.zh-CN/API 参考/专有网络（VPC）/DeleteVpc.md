# DeleteVpc {#reference_i4w_xmt_ndb .reference}

删除一个专有网络（VPC）。

调用该接口创建VPC时，请注意：

-   删除VPC之前，需要先释放或移走VPC内的所有资源，包括交换机，云产品实例，路由器接口，HAVIP等。

-   只有处于Available状态的VPC才可以被删除。
