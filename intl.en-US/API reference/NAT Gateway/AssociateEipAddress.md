# AssociateEipAddress {#reference_i4w_xmt_ndb0 .reference}

Bind an EIP to a cloud product instance in the same region.

**Note:** 

-   You can bind an EIP to an ECS instance of the VPC network, an SLB instance of the VPC network, or a NAT gateway. This document introduces how to use this API to bind an EIP to a NAT gateway. For more information, see [AssociateEipAddress](reseller.en-US/API reference/EIP/AssociateEipAddress.md#).
